# 오디오 파일 임포트 → 분석 버튼 → 결과 확인 (초보자용 MVP)

요청하신 것처럼, 이 프로젝트는 **오디오 파일을 올리고 분석 버튼을 누르면 결과가 바로 나오는** 웹앱 MVP입니다.

## 무엇이 나오나요?
- 길이(초)
- 예상 BPM
- 예상 Key
- 간단 코드 진행
- 멜로디 노트(상위 일부)
- 결과 JSON 다운로드

> ⚠️ 현재 버전은 브라우저(Web Audio API) 기반 간이 분석기입니다. 정밀한 채보(MIDI/PDF 자동 생성)는 서버 모델(예: librosa + music21 + MuseScore) 연동이 추가로 필요합니다.

## 실행 방법 (진짜 초보자용)

### 1) 프로젝트 폴더로 이동
```bash
cd /workspace/codex
```

### 2) 웹서버 실행 (파이썬 기본 내장)
```bash
python -m http.server 8000
```

### 3) 브라우저 열기
아래 주소로 접속:

- http://127.0.0.1:8000

### 4) 사용
1. 파일 선택
2. **분석하기** 클릭
3. 결과 확인
4. 필요하면 **결과 JSON 다운로드** 클릭

## 파일 구성
- `index.html`: 화면(UI)
- `styles.css`: 스타일
- `app.js`: 오디오 분석 로직
